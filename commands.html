Live Update Commands

Live updates are published by talking to an IRC bot on Freenode.

Getting set up on Freenode

Freenode is an IRC chat network. We can use it to talk to each other about moderation and live updates, as well as talk to a bot that will publish live updates. To connect to Freenode, you should have a client:

  • HexChat for Windows
  • Colloqy for OS X
  • WeeChat for Linux (HexChat if you want a GUI)

Pick your client and tell it to connect to Freenode (your client should have Freenode set up automatically, but if not, it's chat.freenode.net).

Each IRC network (like ours, Freenode) consists of several IRC channels. These are chat rooms, with several people chatting in a single channel. The Freenode channel for discussing TPP is called #twitchplayspokemon, and we use a private channel called #tppmods to publish live updates and talk with each other. To join a channel, type /join #channelname. You'll need to be fully set up before you can join #tppmods, though.

Speaking of getting fully set up, we need to establish who you are. Basically, we can't have any old person updating the site - it'll turn into a spamfest right away. To do so, you'll need to register (it's easy): /msg NickServ help register. Once that's out of the way, you'll need an "unaffiliated cloak", which protects your IP and makes sure the bot can consistently identify you (also pretty easy). Type /join #freenode and ask: "Can I get an unaffiliated cloak?". You can mention one of the people with + or a dot next to their names if no one is paying attention.

Once you're identified and cloaked, you're good to go. You need to use /msg NickServ identify password with your password whenever you connect to the network to make sure you're logged in. You can make your life easier by looking around in your IRC settings for the server username and password and putting them in there instead. Send a message to SirCmpwn (/msg SirCmpwn your message here) and ask to be invited to #tppmods.

Talking to the bot

To talk to the bot, just say one of these commands in any channel PokeBot is in (includes #tppmods). You can also send these to PokeBot in a private message (like /msg PokeBot .levelup Pikachu).

  • .update category "title" "description" time: Categories are Travel, Roster, Items, Battle, Commentary, Meta. Time and description optional, default time is the time the update was published.
  • .levelup pokemon: Levels up a Pokemon.
  • .catch "species" level "in-game name" "nickname": Adds a Pokemon. Do not use this if the Pokemon is immediately boxed. Add moves with:
  • .learn pokemon "move": Adds a move to a Pokemon's moveset.
  • .forget pokemon "move": Forgets a move.
  • .release pokemon: Also used for boxing Pokemon.
  • .pickup "item": Adds this item to our inventory.
  • .drop "item": Drops/uses an item.
  • .goal "goal": Updates the goal.
  • .badge "name": Acquires a badge.
  • .money amount: Money.

Times are noted in the form of "1d1h1m1s"

Remember to use the quotes where required!

Examples

You want to make a comment on how the battle's going down: .update commentary "Bird Jesus poisoned" "Bird Jesus is in some trouble, let's hope he pulls through"

This sets the title of the update to "Bird Jesus poisoned", and the extended commentary is "Bird Jesus is in some trouble..."

Say we catch a pidgey: .update roster "Red catches a Pidgey". This doesn't include an extended description - it's optional.

If you want to add an event in the past, try something like: .update battle "Red vs Blue, Blue wins" "Description goes here" 4d5h3m10s

This will add the event at 4 days, 5 hours, 3 minutes, and 10 seconds from the start of Twitch Plays Pokemon (aka the timer shown on the stream).

Notes

If we use or drop an item, use .drop "item name" to indicate that change. Also, use .catch only when Pokemon are added to the party, and .release when Pokemon are removed from the party. Don't use it when we catch a Pokemon and send it to the PC. You should use .catch and .release when we move Pokemon to/from the PC. It updates the party on the front page.

One last thought: the only command that adds an update is .update. The rest modify the other parts of the live update page, like the inventory.

Update Categories

In an effort to organize things, we're trying to use each category for these things:

  • Travel: Moving between routes, or cities, or dungeons (just an area change)
  • Battle: The outcome of battles. Use commentary for updates for each battle
  • Roster: Changes to the Pokemon we own. Released or caught.
  • Items: Pick up, use, or drop items.
  • Commentary: Live commentary, similar to pro sports commentary
  • Meta: Important meta news, forced to show up even if the user turns off all other updates
